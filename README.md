# PET LOVE SHOP & CARE
<h4>Focado na Criação de Cadastros e Perfil de Usuário(Animal com suas Particularidades)</h4>
<ul>
  <li>Criação de Cadastro</li>
  <li>Cadastro Cliente/PET</li>
  <li>Orçamento/Formas de Pagamento</li>
</ul>

Esse é um projeto simples feito em Java para praticar, no momento focando mais no uso de métodos estáticos. </br> Nele consiguirei cadastrar um animal, com todas as particularidades(Raça, Data de Nascimento, ID, etc.).

<i>Não usei arrays, APIs ou recursos avançados — só o básico até o momento.</i>
