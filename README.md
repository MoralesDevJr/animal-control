# Animal Control

Sistema de Cadastro de Animais

Esse é um projeto simples feito em Java para praticar o uso de métodos estáticos. Ele permite cadastrar um animal com nome e senha, e dá a opção de cancelar o cadastro durante o processo.

Não usei arrays, APIs ou recursos avançados — só o básico que aprendi até agora: leitura de dados com Scanner, métodos estáticos e estruturas condicionais.

Estou fazendo isso como parte dos meus estudos em programação Java, focado em fixar conceitos fundamentais.
